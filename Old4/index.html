LowPy

A High-Level Simulator of Neuromorphic ML Processor Non-Idealities

The field of Neuromorphic Computing aims to develop ASIC, hardware-style implementations of machine learning algorithms, among other things. Often, this is accomplished with novel devices, like the memristors pictured above. While these memristors make excellent candidates for storing neural network weights, and even conducting in-memory computing operations, they have their own "side effects", such as write variability, drift, decay, limited precision, and more non-ideal characteristics which can cause weights to behave unexpectedly. The question then becomes: How can one determine if a novel device is suitable for hardware implementation of a particular machine learning algorithm? That's where LowPy comes in.

LowPy is a GPU-accelerateable Python machine learning package which wraps a non-ideal simulator around a Keras model for controlled weight alteration during machine learning fitting, evaluation, and validation.

To install LowPy, simply run

pip install lowpy

in the Terminal/Command prompt of any computer running Python 3. Visit the LowPy GitHub page or the LowPy documentation page for more information.
